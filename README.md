# 花かご Hanakago — Website

A static site for 花かご (Hanakago)

## Deploy to GitHub Pages

1. Create a new **public** repository on GitHub (e.g. `hanakago`).
2. Upload **all the files in this folder** to the root of that repository (drag-and-drop on github.com works).
3. In **Settings → Pages**, set:
   - Source: **Deploy from a branch**
   - Branch: **main** / **/ (root)**
4. Save. Your site will be live at `https://<your-username>.github.io/<repo-name>/` in 1–2 minutes.

## Local preview

Just open `index.html` in any browser — no build step needed.
